# 数据库枚举值与状态码映射

## 1. 订单状态 (t_orders.status)
数据库中使用英文字符串表示状态，具体含义如下：
- `PENDING`: **待支付**。用户已下单但未付款。
- `PAID`: **已支付**。用户完成付款，等待发货。
- `SHIPPED`: **已发货**。物流运输中。
- `COMPLETED`: **已完成**。用户确认收货。
- `CANCELLED`: **已取消**。用户手动取消或超时自动关闭。
- `REFUNDED`: **已退款**。售后流程结束。

## 2. 商品分类 (t_products.category)
目前的标准分类包括：
- '电子产品': 包含手机、电脑、键盘、鼠标等数码设备。
- '家具': 包含椅子、桌子、床等。
- '图书': 包含各类纸质书籍。
- '服装': 包含上衣、裤子、鞋帽。

## 3. 用户等级 (t_users.level)
- `普通会员`: 注册后的默认等级。
- `VIP`: 年消费满 1000 元自动升级。
- `SVIP`: 年消费满 10000 元自动升级。
